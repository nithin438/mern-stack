What is Express?

Express is a Node.js framework that makes it easy to build web servers.

How HTML Rendering Works

  1. User opens a URL in the browser
  2. Browser sends a request to the server
  3. Express server finds the matching route
  4. Server sends back the HTML file
  5. Browser displays the page
